# Todo Uai

Gerenciador de tarefas simples, moderno e sem enrolação. Do jeitim mineiro de fazer as coisas.

## Tecnologias

Este projeto foi desenvolvido com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como executar

```sh
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview da build de produção
npm run preview
```

## Desenvolvimento

O projeto utiliza:
- **React Router** para navegação
- **TanStack Query** para gerenciamento de estado
- **shadcn/ui** para componentes UI
- **Tailwind CSS** para estilização
